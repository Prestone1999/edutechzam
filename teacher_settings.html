Student Dashboard Settings

Clear Latest Updates

This will clear the 'Latest Test', 'Latest Assignment', and 'Latest Announcement' sections on the student dashboard.

Clear All Courses

This will remove all courses from the 'My Courses' section on the student dashboard.

Erase All Tests

This will permanently delete all created tests. This action cannot be undone.

Clear Announcements

This will remove all announcements from the system.

Clear Grades

This will remove all grades from the system.

Clear Messages

This will remove all messages from the system.

Clear Assignments

This will remove all assignments from the system.

Clear All Data

This will remove all tests, announcements, grades, messages, and assignments. This action cannot be undone.
